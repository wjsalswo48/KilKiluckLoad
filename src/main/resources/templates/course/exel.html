주의 사항!!


1. 새로운 코스에 대해서 쓰고 싶다면 코스정보   ->    길 정보   ->   주변시설(관광지, 화장실, 식당 등등) 순서로 엑셀파일을 제출해주세요.

2. 각 엑셀 파일의 코스, 구간, seq, 코드값은 건드리지 마세요 -> 만약 실수로 건드리시면 빠르게 다시 모든 정보 가져오기(정보 가져오기) 버튼을 클릭해주세요 초기상태로 돌려줍니다.

3. 사용여부는 데이터베이스에서 삭제를 시킵니다 (use/delete만 입력하세요) -> 삭제 후 엑셀 파일에서 지우시면 되돌리기 어려우니 주의 하셔서 사용해주세요.

4. 하나의 행의 유효한 열을 비우지 말아 주세요. (채울내용이 없으면 null이라고 채워주세요)

5. 등록하려는 정보의 코스 및 구간이 존재하는지 확인 하세요.

6. 코스 및 등록
              -->    ex : 코스 ( 1코스  ->  01C(코드값) ), 구간 ( 1코스 1구간  ->  01C 01G(코드값) )

엑셀 파일만 올려 주세요
